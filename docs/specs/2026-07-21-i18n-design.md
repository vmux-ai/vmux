# Internationalization

## Scope

- Bundle 115 complete Fluent locale tags: English, Japanese, regional variants, and broad ISO 639-1 coverage.
- Accept any valid BCP 47 locale from `appearance.locale`.
- Default to the operating-system locale when set to `system`.
- Fall back per message to English.
- Apply Unicode text direction to every vmux page.
- Support locale packs at `~/.vmux/locales/<tag>.ftl` without rebuilding vmux.

## Runtime

The host resolves the requested locale and sends it with theme state to each webview. Embedded or external Fluent catalogs format messages, variables, and plural selectors. Webviews set the root `lang` and `dir` attributes. Native tray labels use the same catalog runtime.

Locale lookup order:

1. Exact external tag, such as `pt-BR.ftl`.
2. Base external language, such as `pt.ftl`.
3. Exact bundled tag.
4. Bundled base language.
5. Bundled English.

## Catalog contract

English owns message IDs and variables. Bundled catalogs have exact message parity; catalogs beyond Japanese are machine translated. External catalogs may be partial because missing messages fall back to English. Invalid external catalogs are ignored without replacing the last valid catalog.
