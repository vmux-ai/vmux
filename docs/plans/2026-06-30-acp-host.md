# ACP Host (Milestones B + C) Implementation Plan

> **For agentic workers:** This plan is executed **inline and directly** in one session (NOT subagent-driven — vmux CEF builds are large and long-lived subagents drop sockets; see the project's "subagent CEF build fragility" rule). Warm the target dir with a background `cargo build` first, then iterate incrementally. Runtime testing is **one pass at the end** (the user runs it), per the project's "finish then test" rule. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** vmux implements the ACP (Agent Client Protocol) `Client` role so any external ACP agent (claude-code-acp / gemini / vibe-acp) plugs into vmux's native panes — streaming chat, permission prompts, fs diffs, real terminals — plus the vmux_mcp toolset (scope C).

**Architecture:** The ACP client is hosted **in the `vmux_service` daemon** (where subprocesses, PTYs, and the tokio runtime already live), mirroring `AgentSessionManager::run_session`. ACP `session/update` notifications are projected to the existing `Message` shape and emitted as the existing `ServiceMessage::AgentDelta`/`AgentMessagesSnapshot`/`AgentAwaitingApproval`/`AgentRunStatusChanged`, so the existing GUI pump (`vmux_terminal` drain → `consume_page_agent_stream`) and `approval.rs` are reused verbatim. The GUI side adds `AgentVariant::Acp` mirroring the Page path. Terminals map to real `vmux_terminal` panes; fs edits surface via a new `vmux_editor` proposed-edit overlay; scope-C tools reuse the existing `vmux mcp --anchor` stdio sidecar with terminal tools omitted.

**Tech Stack:** Rust, Bevy ECS, tokio, `agent-client-protocol = "1.0.1"` (+ `agent-client-protocol-schema` 1.1.0 via re-export), `tokio_util::compat`. No `agent-client-protocol-tokio` (pinned to 0.11, incompatible) and no `agent-client-protocol-rmcp`.

**Spec:** `docs/specs/2026-06-30-acp-host-design.md`. Read it before starting.

## Conventions (read once)

- **Worktree:** all edits happen in `.worktrees/acp-host` (branch `feat/acp-host`). Use worktree-prefixed paths.
- **Module pattern:** filename-based, no `mod.rs` (e.g. `acp.rs` + `acp/` dir).
- **Bevy `App` builder:** chain consecutive builder calls in one expression; register a plugin's written messages in `build()` (idempotent), not per-test.
- **No explanatory comments** in code; rustdoc `///` allowed.
- **Native tests only during the loop** (`cargo test -p <crate>`); no `make dev`. Warm build: `bash -c "cd .worktrees/acp-host && cargo build -p vmux_service -p vmux_agent -p vmux_editor 2>&1 | tail -20"` in the background first.
- **After `cargo fmt`:** `git checkout -- patches/` (fmt reformats vendored crates).
- **acp API note:** all acp types are verified from crate source, but the **handler-registration closure shape** (`.on_receive_request(closure, on_receive_request!())`) is the one thing to confirm against the compiler — Task 1 locks it. Everything downstream uses the shape Task 1 proves.

```rust
    pub fn messages(&self) -> &[Message] { &self.messages }

    pub fn apply(&mut self, update: SessionUpdate) -> Vec<Intent> {
        match update {
            SessionUpdate::AgentMessageChunk(c) => self.append_text(c, false),
            SessionUpdate::AgentThoughtChunk(c) => self.append_text(c, true),
            SessionUpdate::UserMessageChunk(_) => vec![], // skip echo of locally-authored prompt

            SessionUpdate::ToolCallUpdate(u) => self.update_tool_call(u),
            SessionUpdate::Plan(p) => self.upsert_plan(p),
            _ => vec![], // PlanUpdate/PlanRemoved/capabilities/usage: B = ignore/log

    // update_tool_call: patch status/content of the existing block by id.
```

```rust
use tokio::process::Command;
use tokio_util::compat::{TokioAsyncReadCompatExt, TokioAsyncWriteCompatExt};

pub struct Spawned {
    pub child: tokio::process::Child,
    pub transport: agent_client_protocol::ByteStreams< /* writer */, /* reader */ >,
    pub stderr: tokio::process::ChildStderr,
}

pub fn spawn(command: &str, args: &[String], env: &[(String, String)], cwd: &std::path::Path)
    -> std::io::Result<Spawned>
{
    let mut cmd = Command::new(command);
    cmd.args(args).envs(env.iter().cloned()).current_dir(cwd)
        .stdin(Stdio::piped()).stdout(Stdio::piped()).stderr(Stdio::piped());
    let mut child = cmd.spawn()?;
    let stdin = child.stdin.take().unwrap().compat_write();
    let stdout = child.stdout.take().unwrap().compat();
    let stderr = child.stderr.take().unwrap();
    let transport = agent_client_protocol::ByteStreams::new(stdin, stdout);
    Ok(Spawned { child, transport, stderr })
}
```

```rust
use agent_client_protocol::schema::v1::*;
use agent_client_protocol::Client;
use tokio::sync::{mpsc, broadcast, oneshot};
use crate::protocol::{ServiceMessage, AgentRunStatus};

pub enum AcpInput {
    User(String),
    Approve { call_id: String, decision: crate::protocol::ApprovalDecision },
    Close,
}

pub struct AcpShared {
    pub sid: String,
    pub anchor: vmux_core::ProcessId,
    pub stream_tx: broadcast::Sender<ServiceMessage>,
    pub terminals: std::sync::Mutex<std::collections::HashMap<String, vmux_core::ProcessId>>,
    pub pending_perms: std::sync::Mutex<std::collections::HashMap<String, oneshot::Sender<crate::protocol::ApprovalDecision>>>,
    pub cwd: std::path::PathBuf,
    pub projector: std::sync::Mutex<super::projector::AcpProjector>,
}

pub async fn run(
    cfg_command: String, cfg_args: Vec<String>, cfg_env: Vec<(String, String)>,
    shared: std::sync::Arc<AcpShared>,
    mut input_rx: mpsc::UnboundedReceiver<AcpInput>,
    mcp_servers: Vec<McpServer>,
) {
    let spawned = match super::transport::spawn(&cfg_command, &cfg_args, &cfg_env, &shared.cwd) {
        Ok(s) => s,
        Err(e) => { emit_errored(&shared, format!("spawn failed: {e}")); return; }
    };
    // drain stderr to the log
    let stderr = spawned.stderr;
    tokio::spawn(async move { log_stderr(stderr).await; });

    let result = Client.builder()
        // handlers (Tasks 6-8). Use the Task-1 macro form:
        .on_receive_request(super::handlers::permission(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::read_file(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::write_file(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::terminal_create(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::terminal_output(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::terminal_wait(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::terminal_kill(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_request(super::handlers::terminal_release(shared.clone()), agent_client_protocol::on_receive_request!())
        .on_receive_notification(super::handlers::session_update(shared.clone()), agent_client_protocol::on_receive_notification!())
        .connect_with(spawned.transport, {
            let shared = shared.clone();
            |cx| async move {
                let mut init = InitializeRequest::new(ProtocolVersion::V1);
                init.client_capabilities = caps();
                cx.send_request(init).block_task().await?;
                let mut ns = NewSessionRequest::new(shared.cwd.clone());
                ns.mcp_servers = mcp_servers;
                let s = cx.send_request(ns).block_task().await?;
                set_session_id(&shared, s.session_id.clone());
                emit_status(&shared, AgentRunStatus::Idle);
                while let Some(input) = input_rx.recv().await {
                    match input {
                        AcpInput::User(text) => {
                            emit_status(&shared, AgentRunStatus::Streaming);
                            let r = cx.send_request(PromptRequest::new(
                                s.session_id.clone(),
                                vec![ContentBlock::Text(TextContent::new(text))],
                            )).block_task().await;

                            }

                        AcpInput::Close => {
                            cx.send_notification(CancelNotification::new(s.session_id.clone()));
                            break;
                        }
                    }
                }
                Ok(())
            }
        }).await;
    if let Err(e) = result { emit_errored(&shared, format!("acp connection ended: {e}")); }
}
```

```rust
    pub fn spawn(&self, sid: String, command: String, args: Vec<String>,
                 env: Vec<(String,String)>, cwd: std::path::PathBuf,
                 stream_tx: broadcast::Sender<crate::protocol::ServiceMessage>,
                 mcp_servers: Vec<agent_client_protocol::schema::v1::McpServer>) {

            None => false,
        }

    pub fn contains(&self, sid: &str) -> bool { self.sessions.lock().unwrap().contains_key(sid) }
    pub fn close(&self, sid: &str) {
        if let Some(h) = self.sessions.lock().unwrap().remove(sid) {
            let _ = h.input_tx.send(AcpInput::Close);

    }
```
